## 介绍
cc-project-easyui 是一个传统的后端项目，页面使用是jsp+jquery+easyui；后端使用的是srping mvc +mybatis；管理工具使用的是maven；数据库采用的是mysql。
这套权限平台是angular和vue版本的前生，大部分代码还是和这套的代码差不多。
